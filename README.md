Project Overview

Developed an end-to-end customer churn analytics pipeline using SAS to identify factors influencing customer attrition and support data-driven retention strategies. The project includes data exploration, preprocessing, feature engineering, statistical analysis, and predictive modeling on customer behavioral and subscription data.

Dataset

The dataset contains customer demographic information, subscription details, spending behavior, service usage patterns, support interactions, and churn status.

Key Tasks
1. Performed Exploratory Data Analysis (EDA) to understand customer behavior and churn patterns
2. Conducted data quality assessment, including missing value detection and duplicate record analysis
3. Generated descriptive statistics, frequency distributions, correlation analysis, and visualizations
4. Applied data cleaning techniques, including missing value imputation and categorical variable encoding
5. Engineered new predictive features such as:
Support Call Rate
Spending Per Tenure
Payment Risk Index
Age Group Segmentation
6. Standardized numerical variables to improve model performance
7. Split the dataset into training and testing sets for model evaluation
8. Built and evaluated predictive churn models using SAS

Key Insights
1. Identified customer behavior patterns associated with higher churn probability
2. Revealed relationships between support interactions, payment delays, spending habits, and customer retention
3. Developed engineered features that improved customer risk profiling and predictive analysis
